data Token #

Holds all valid tokens that can be used in a program. Each token has the name of the corresponding operator or constant followed by the suffix Tok

lexer :: String -> [Token] #

Translates a program in a string to a list of corresponding valid tokens.

If an invalid token is found an error is displayed.

getWordToken :: String -> Token #

Transforms a word into its corresponding token. The word may be one of the reserved keywords ("not", "and", etc.) or a valid name for a variable, that is, a word that starts with an undercased letter followed by any number of letters, digits and underscores (_).
